# LENOVO ThinkCentre

## Bypass whitelist

Download `USB Drive UEFI BIOS Flash Package` and Boot enter the DOS environment

Run the commands:

```plain
flash2.exe /sn:INVALID
flash2.exe /mtm:INVALID
```
